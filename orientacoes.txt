# Orientações para o Exercício de HTML e CSS

1. Analise todo HTML existente nos dois arquivos fornecidos

É importante compreender a estrutura HTML destas páginas ANTES de começar a fazer o CSS

2. Crie uma pasta "css" e dentro dela crie o arquivo "estilos-exercicio.css".

3. Conecte o arquivo "estilos-exercicio.css" aos arquivos "index.html" e "web.html"

4. Comece a desenvolver o CSS livremente visando realizar uma boa estilização das páginas.

Tente utilizar diversos dos recursos que vimos até agora, como diferentes tipos de regras e seletores e diversas propriedades/valores.
